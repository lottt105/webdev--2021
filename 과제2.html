기초웹개발론


IntelliJ IDEA 단축키

  • Alt + Enter : 원하는 부분에 커서를 놓고 alt + enter를 누르면 오류에 대한 수정 제안 목록이 표시
  • F2 : 다음 오류, 경고, 제안으로 이동
  • Alt + 1(Win/Linux), Cmd + 1(Mac) : 프로젝트 창을 열고 초점을 이동
  • Esc : 프로젝트 창에서 다시 에디터 창으로 초점을 이동, 변경 사항을 저장하지 않고 모든 팝업을 닫는 데 유용
  • Ctrl + E(Win/Linux), Cmd + E(Mac) : 최근 파일 대화상자가 열려 방향키로 탐색 가능
  • Ctrl + B(Win/Linux), Cmd + B(Mac) : 심볼의 선언으로 이동
    -> Ctrl + Alt+ B(Win/Linux), Alt + Cmd + B(Mac) : 구현 탐색
  • Alt + F7 : 사용된 모든 위치 표시
  • Tab Ctrl Twice : Run Anything 창이 열림
  • Ctrl + W/Ctrl + Shift + W(Win/Linux), Alt + 상/하 방향키(Mac) : 선택 영역 확장
  • Ctrl + /(Win/Linux), Cmd + /(Mac) : 커서가 있는 줄 또는 선택되 영역을 주석 처리
  • Shift + Ctrl + Enter(Win/Linux), Shift + Cmd + Enter(Mac) : 구문 완성
  • Ctrl + Alt+ L(Win/Linux), Alt + Cmd + L(Mac) : 현재 파일의 서식을 프로젝트 표준에 맞춤
  • Shift + Ctrl + Alt+ T(Win/Linux), Ctrl + T(Mac) : 모든 리펙토링을 표시
  • Shift + Ctrl + A(Win/Linux), Shift + Cmd + A(Mac) : IntelliJ IDEA의 모든 액션 검색 가능
  • Tab Shift Twice : 전체 검색 창 표시
